# Personal CV Website Template

This repository has been cleaned up from the original author's blog content, media files, and profile details so it can be reused as a personal CV or portfolio site.

## What Was Removed

- blog posts and their images
- RSS and search pages
- newsletter and post-sharing components
- original profile references, metadata, and social links

## What You Still Need To Replace

- name and site metadata in `src/consts.ts`
- social links in `src/constants.ts`
- homepage content in `src/pages/index.astro`
- about page content in `src/pages/about.mdx`

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
