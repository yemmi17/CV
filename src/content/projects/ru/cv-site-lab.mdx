CV Site Lab - это мой личный сайт-резюме, который я развиваю не как разовую HTML-страницу, а как небольшой  инженерный продукт. В нем должны быть понятная структура профиля, проекты, блог, bilingual-контент, SEO и предсказуемый процесс сборки.

Главная идея проекта - показать не только итоговое резюме, но и сам инженерный подход: как принимаются решения, как ограничивается scope, как проверяются изменения и как сайт постепенно становится удобным артефактом для найма, обучения и публичной технической практики.

## Инженерный фокус проекта

- Static-first сайт на Astro с контентом в MDX и Astro content collections.
- Разделение профиля, проектов и блога, чтобы каждая часть сайта имела понятную ответственность.
- Двуязычная структура страниц: сначала качественный русский контент, затем адаптированная английская версия.
- SEO-гигиена: canonical URL, alternate links, structured data и аккуратные описания страниц.
- Проверяемый workflow: маленькие диффы, code review-friendly изменения, npm run check и production build перед завершением задач.

## Ближайшее развитие

- Наполнить блог первыми заметками о создании сайта.
- Довести русскую и английскую версии проекта до одинакового уровня качества.
- Усилить CI/CD-проверки и документировать процесс деплоя.
- Постепенно добавлять реальные project notes вместо абстрактных описаний.
