Проект описывает практичный базис платформы для запуска content-heavy сервиса в AWS.

Фокус не на одном инструменте, а на полном эксплуатационном пути: provisioning инфраструктуры, Kubernetes workloads, поведение autoscaling, observability и предсказуемые deployment-процессы.

## Инженерный фокус

- Infrastructure as code через Terraform.
- Управление Kubernetes workload и autoscaling.
- Метрики и dashboards для повседневной эксплуатации.
- Понятные deployment-границы и изменения, которые удобно откатывать.
