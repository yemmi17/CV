A reusable GitOps baseline for deploying services to Kubernetes with Argo CD and Helm.

The goal is to keep delivery predictable: configuration is versioned, deployment state is visible, and routine changes can be reviewed before they reach the cluster.

## Engineering Focus

- GitOps delivery model with Argo CD.
- Helm chart structure for repeatable deployments.
- CI checks before configuration reaches production.
- Clear separation between application code and environment configuration.
