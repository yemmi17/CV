This project describes a practical platform baseline for running a content-heavy service in AWS.

The focus is not on a single tool, but on the full operational path: infrastructure provisioning, Kubernetes workloads, scaling behavior, observability, and predictable deployment routines.

## Engineering Focus

- Infrastructure as code with Terraform.
- Kubernetes workload management and autoscaling.
- Metrics and dashboards for day-to-day operations.
- Clear deployment boundaries and rollback-friendly changes.
